# Project Overview — Wireshark Fundamentals Lab

## 🎯 Purpose of This Lab
This project introduces the fundamentals of packet capture and network traffic analysis using **Wireshark**, the industry-standard tool used by network engineers, SOC analysts, and IT professionals.  
The lab is designed to help you understand how network packets flow, how to capture them, and how to interpret protocol behavior in real time.

By completing this lab, you will learn how to:
- Install and configure Wireshark
- Capture live network traffic
- Identify active network interfaces
- Observe packet flow in real time
- Generate traffic for analysis
- Begin interpreting packet metadata (protocols, source/destination, length, info)

---

## 🧱 What This Lab Covers
This lab focuses on the foundational skills required to use Wireshark effectively:

### **1. Installing Wireshark**
- Downloading the correct installer for your OS  
- Understanding optional components (WinPcap/Npcap on Windows)

### **2. Running Your First Packet Capture**
- Selecting the correct network interface  
- Starting and stopping captures  
- Observing live packet flow  
- Generating traffic by browsing websites  

### **3. Understanding the Wireshark Interface**
- Packet list pane  
- Packet details pane  
- Packet bytes pane  
- Protocol color coding  

### **4. Basic Traffic Interpretation**
- Identifying common protocols (DNS, TCP, HTTP, TLS)  
- Understanding packet direction and flow  
- Recognizing normal vs. unusual traffic patterns
